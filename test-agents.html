How to Test Agent-to-Agent Conversations

  1. Open the MACP iOS app on your device
  2. Make sure you have at least one personal agent configured
  3. Ensure you have an API key set in Settings
  4. Scan any QR code below with your phone's camera
  5. Choose "Send My Agent" to start an autonomous conversation
  6. Watch your agent converse with the test agent in real-time!
