Token Economy

Each user message triggers multiple LLM calls: orchestration analysis (which agents should respond), memory retrieval (inject relevant context), agent response generation, and background fact extraction. Multi-agent responses multiply costs but can be optimized.

Optimization levers: Cache memory summaries, batch extraction, limit parallel agents, use cheaper models for orchestration.

Path to Profitability

LLM costs dominate (86% at scale). This is actually good news - LLM costs are dropping 10x every 18 months. Infrastructure costs are largely fixed and amortize well. As LLM prices continue falling, margins expand automatically.

Revenue model: Freemium with message limits. Free tier: 20 msgs/day. Pro: $9.99/mo unlimited. Business: $49/mo per seat. API access: usage-based with 25% markup.

Key Assumptions

Model Assumptions

Usage: Average user sends 75 messages/month (2.5/day). Power users 200+, casual users 20-30.

LLM costs: Based on Claude Sonnet 4 pricing ($0.003 input, $0.015 output per 1K tokens). Average message = 800 input tokens (user + memory + system) + 300 output tokens.

Orchestration overhead: +40% token usage for multi-agent routing and background extraction.

Infrastructure: AWS us-east-1 pricing. Aurora Serverless v2 with auto-scaling. Reserved instances would reduce costs 30-40% at scale.

Risks: LLM price increases (unlikely given competition), usage spikes beyond projections, data storage growth.

Upside: LLM prices continuing to fall, enterprise contracts with higher margins, marketplace transaction fees.

Not included: Personnel costs, marketing spend, legal/compliance, office space.
